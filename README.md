# dotfiles

## Prerequisites

1. Install Homebrew.
2. Install Ansible (`brew install ansible`).
3. Install [Go](http://golang.org/dl/).

## Installing

```
git clone git@github.com:stuartrexking/dotfiles.git
cd dotfiles
make
```
